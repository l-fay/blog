手里有一个1920*1080的视频，想把它裁成1920*800的视频。

<!-- more -->

主要使用到了视频滤镜`-vf`的`crop`参数。

```
crop的参数格式为w:h:x:y，
w、h为输出视频的宽和高单位是像素，
x、y标记输入视频中的某点，将该点作为基准点，向右下进行裁剪得到输出视频，单位也是像素。
如果x y不写的话，默认居中剪切
```

完整命令为：

```
ffmpeg -i 1.mp4 -vf crop=1920:800:0:140 2.mp4
```
